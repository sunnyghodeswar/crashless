# Image Assets

This directory contains image assets for the Crashless documentation site.

## Required Images

### Logo Files
- `logo.png` - Light mode logo (32x32px recommended)
- `logo-dark.png` - Dark mode logo (32x32px recommended)
- `favicon.ico` - Browser favicon (16x16px or 32x32px)

### Banner
- `banner.png` - Main banner image for README (1200x400px recommended)

### Social Card
- `crashless-social-card.jpg` - Social media preview image (1200x630px recommended)

## Design Guidelines

- **Logo**: Simple, recognizable, works at small sizes
- **Banner**: Eye-catching, represents the product, includes text if needed
- **Social Card**: Professional, includes logo and tagline
- **Favicon**: Simplified version of logo
